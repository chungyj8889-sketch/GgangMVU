# 변수 프롬프트

변수를 활용한 프롬프트 작성 방법을 알아봅니다.

## 개요

변수 프롬프트를 사용하면 동적인 콘텐츠를 생성할 수 있습니다. 변수 값에 따라 프롬프트 내용이 자동으로 변경됩니다.

## 기본 문법

### 단일 변수 삽입

```
{{변수명}}
```

### 조건부 표현

```
{{#if 조건변수}}
  조건이 참일 때 표시할 내용
{{/if}}
```

### 반복문

```
{{#each 배열변수}}
  {{this}}
{{/each}}
```

## 실제 예시

### 캐릭터 소개 프롬프트

```
이름: {{이름}}
나이: {{나이}}세
직업: {{직업}}

{{이름}}은(는) {{성격}}한 성격을 가지고 있습니다.
```

!!! example "결과 예시"
    이름: 김철수  
    나이: 25세  
    직업: 프로그래머  
    
    김철수은(는) 활발한 성격을 가지고 있습니다.

## 고급 활용

### 중첩 변수

변수 안에 다른 변수를 참조할 수 있습니다:

```
{{캐릭터.이름}}의 {{캐릭터.특성}}
```

### 기본값 설정

변수가 없을 경우 기본값을 지정할 수 있습니다:

```
{{변수명 | default: "기본값"}}
```
