# 변수 구조 작성

효율적인 변수 구조를 설계하고 작성하는 방법을 알아봅니다.

## 좋은 변수 구조의 특징

- **명확한 명명**: 변수의 용도를 알 수 있는 이름
- **계층적 구조**: 관련 변수를 그룹화
- **일관된 규칙**: 통일된 명명 규칙 사용

## 변수 명명 규칙

### 권장 규칙

| 유형 | 규칙 | 예시 |
|------|------|------|
| 캐릭터 속성 | `캐릭터_속성` | `주인공_HP`, `주인공_이름` |
| 시스템 변수 | `SYS_이름` | `SYS_시간`, `SYS_날씨` |
| 임시 변수 | `TEMP_이름` | `TEMP_계산값` |
| 상수 | `CONST_이름` | `CONST_최대HP` |

### 피해야 할 것

!!! warning "피해야 할 명명"
    - 한 글자 변수명: `a`, `x`
    - 모호한 이름: `데이터`, `값`
    - 공백이 포함된 이름: `캐릭터 이름`

## 계층적 구조 예시

```javascript
{
  "캐릭터": {
    "주인공": {
      "기본정보": {
        "이름": "철수",
        "나이": 25
      },
      "스탯": {
        "HP": 100,
        "MP": 50,
        "공격력": 20
      },
      "인벤토리": []
    }
  },
  "월드": {
    "현재위치": "마을",
    "시간": 12,
    "날씨": "맑음"
  }
}
```

## 변수 그룹화

관련 변수들을 그룹으로 묶어 관리하면 편리합니다:

```javascript
// 전투 관련 변수 그룹
const 전투변수 = {
  "전투중": false,
  "턴": 0,
  "적목록": []
};
```

## 템플릿 활용

자주 사용하는 변수 구조는 템플릿으로 저장해두세요:

```javascript
// 캐릭터 템플릿
const 캐릭터템플릿 = {
  "이름": "",
  "HP": 100,
  "MP": 50,
  "레벨": 1
};
```

!!! tip "버전 관리"
    변수 구조를 변경할 때는 버전을 기록해두면 마이그레이션이 쉬워집니다.
