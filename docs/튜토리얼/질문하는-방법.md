# 질문하는 방법

효과적으로 도움을 받을 수 있는 질문 방법을 안내합니다.

## 좋은 질문의 특징

좋은 질문은 빠르고 정확한 답변을 받을 수 있게 해줍니다.

### 필수 포함 사항

1. **현재 상황**: 무엇을 하려고 했는지
2. **예상 결과**: 어떤 결과를 기대했는지
3. **실제 결과**: 실제로 어떤 일이 발생했는지
4. **시도한 것**: 이미 시도해본 해결책

## 질문 템플릿

```markdown
## 문제 설명
[무엇이 문제인지 간단히 설명]

## 환경
- MVU Zod 버전: 
- 사용 환경: 

## 재현 방법
1. [첫 번째 단계]
2. [두 번째 단계]
3. [문제 발생]

## 예상 결과
[어떤 결과를 기대했는지]

## 실제 결과
[실제로 어떤 일이 발생했는지]

## 시도한 해결책
- [시도 1]
- [시도 2]

## 관련 설정/코드
[관련 설정이나 코드 첨부]
```

## 질문 예시

### ❌ 나쁜 질문

> "안 돼요"
> "변수가 작동 안 해요"
> "왜 이래요?"

### ✅ 좋은 질문

> "HP 변수를 `{{HP}}`로 표시하려고 하는데, 프롬프트에서 값이 아닌 변수 이름 그대로 출력됩니다.
>
> - MVU Zod 버전: 1.2.3
> - 변수 설정: `{"HP": 100}`
> - 사용한 프롬프트: `현재 HP는 {{HP}}입니다`
> - 예상 결과: `현재 HP는 100입니다`
> - 실제 결과: `현재 HP는 {{HP}}입니다`
>
> 변수 설정 패널에서 HP가 정상적으로 등록된 것은 확인했습니다."

## 자주 묻는 질문 확인하기

질문하기 전에 FAQ를 먼저 확인해보세요:

??? question "변수가 인식되지 않아요"
    변수 이름에 공백이 있는지, 중괄호가 올바르게 작성되었는지 확인하세요.
    `{{ 변수 }}`가 아닌 `{{변수}}`여야 합니다.

??? question "스크립트가 실행되지 않아요"
    스크립트 실행이 활성화되어 있는지 설정을 확인하세요.
    `설정 > 고급 > 스크립트 실행 허용`

??? question "마이그레이션 후 오류가 발생해요"
    호환성 모드를 임시로 활성화하고, 오류가 발생하는 부분을 하나씩 수정하세요.

!!! info "응답 시간"
    커뮤니티에서 질문에 대한 응답은 보통 24시간 이내에 제공됩니다.
