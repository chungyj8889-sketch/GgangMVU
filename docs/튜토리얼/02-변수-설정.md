# 변수 설정

MVU Zod에서 변수를 설정하고 관리하는 방법을 알아봅니다.

## 변수란?

변수는 동적으로 변경될 수 있는 값을 저장하는 컨테이너입니다. MVU Zod에서 변수를 활용하면 더욱 유연한 프롬프트 작성이 가능합니다.

## 기본 변수 설정

### 변수 생성

새로운 변수를 생성하려면:

1. 변수 관리 패널을 엽니다
2. "새 변수 추가" 버튼을 클릭합니다
3. 변수 이름과 초기값을 입력합니다

```javascript
// 예시
{
  "변수명": "캐릭터이름",
  "값": "주인공"
}
```

### 변수 타입

| 타입 | 설명 | 예시 |
|------|------|------|
| 텍스트 | 문자열 값 | `"안녕하세요"` |
| 숫자 | 정수 또는 실수 | `42`, `3.14` |
| 불린 | 참/거짓 | `true`, `false` |
| 배열 | 값의 목록 | `["a", "b", "c"]` |

## 변수 사용하기

변수를 프롬프트에서 사용하려면 `{{변수명}}` 형식을 사용합니다:

```
안녕하세요, {{캐릭터이름}}님!
```

!!! tip "변수 자동완성"
    `{{`를 입력하면 사용 가능한 변수 목록이 자동으로 표시됩니다.
