# 플러그인 설치

MVU Zod 플러그인을 설치하는 방법을 안내합니다.

## 사전 요구사항

플러그인을 설치하기 전에 다음 사항을 확인하세요:

- 호환되는 애플리케이션 버전
- 필요한 시스템 요구사항

## 설치 방법

### 방법 1: 직접 설치

1. 플러그인 파일을 다운로드합니다
2. 플러그인 폴더에 파일을 복사합니다
3. 애플리케이션을 재시작합니다

### 방법 2: 마켓플레이스에서 설치

1. 마켓플레이스를 엽니다
2. "MVU Zod"를 검색합니다
3. 설치 버튼을 클릭합니다

## 설치 확인

설치가 완료되면 다음과 같이 확인할 수 있습니다:

```
설정 > 플러그인 > MVU Zod
```

!!! success "설치 완료"
    플러그인이 정상적으로 설치되었다면 다음 단계인 [변수 설정](변수-설정.md)으로 진행하세요.

## 문제 해결

??? question "플러그인이 보이지 않아요"
    애플리케이션을 재시작해보세요. 문제가 지속되면 캐시를 삭제해보세요.

??? question "설치 중 오류가 발생해요"
    버전 호환성을 확인하고, 최신 버전으로 업데이트해보세요.
